# ZiMuZu - Alfred Workflow

*ZiMuZu.tv 字幕组*


## Usage

+ `zmz + keyword` 搜索
+ `zmzh` 24小时下载热门
+ `Enter`、`Cmd + Enter`复制电驴链接 
+ `Alt + Enter` 复制磁力链接
+ `Ctrl + Enter`复制全季所有电驴链接
+ `Shift + Enter`复制全集所有磁力链接
